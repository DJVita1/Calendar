# Календарь на 2026 год

## Описание файлов

- **`index.html`** — основная структура страницы: заголовок, таблица календаря, кнопки навигации.
- **`style.css`** — стили: фон, шрифт, оформление ячеек, выделение выходных и праздничных дней.
- **`script.js`** — логика: генерация календаря на 2026 год, определение праздников, переключение месяцев.
